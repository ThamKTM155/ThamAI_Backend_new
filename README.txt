# ThamAI Backend (Phiên bản miễn phí)
📂 Thư mục: ThamAI_Backend_new

## Endpoint
| Endpoint | Mô tả | Phương thức |
|-----------|--------|-------------|
| /chat     | Chat GPT-4o-mini (thật) | POST |
| /whisper  | Mô phỏng nhận giọng nói | POST |
| /speak    | gTTS tiếng Việt (nam/nữ) | POST |

## Cài đặt
1️⃣ pip install -r requirements.txt
2️⃣ Cập nhật file .env với API key thật.
3️⃣ python app.py

👩 ThạchAI (nữ, Bắc) – mặc định  
👨 ThạchAI (nam, Bắc) – khi đổi giọng trong frontend
